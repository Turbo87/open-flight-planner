Development
------------------------------------------------------------------------------

Install gulp (build tool):

    npm install -g gulp

Install project dependencies:

    npm install

Run the development server (`out/dev/`):

    gulp serve

Run the production build (`out/prod/`):

    NODE_ENV=production gulp
